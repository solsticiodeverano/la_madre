# LA MADRE

TP Final Diseño Digital — MAE UNTREF 2026  
Primer anillo de Telépata: del Big Bang a la creación de las culturas.

## Cómo abrir
Abrir `index.html` en navegador con internet para cargar p5.js desde CDN.  
También se puede reemplazar el script CDN por una copia local de `p5.min.js`.

## Controles
- Scroll / flechas ↑↓: avanzar o retroceder en el viaje.
- Drag mouse: rotar cámara.
- Teclas 1–5: saltar escenas.
- S: guardar captura PNG.

## Estructura
- `index.html`: entrada web.
- `style.css`: interfaz visual.
- `src/utils.js`: funciones comunes.
- `src/scenes.js`: objetos y escenas generativas.
- `src/sketch.js`: timeline, controles y composición.
- `refs/`: códigos fuente originales usados como referencia.
- `docs/LA_MADRE_explicacion.txt`: texto de entrega.
- `capturas/`: imagen de referencia para entrega.
